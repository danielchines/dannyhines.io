---
title: 'My VS Code Settings'
description: 'My settings.json file for Visual Studio Code'
tags: 'vscode'
---

You can use these settings in your workspace by placing them in `./vscode/settings.json`. You can
find your User (global) settings at `~/Library/Application Support/Code/User/settings.json`, which
will be the default settings for all workspaces.

  "editor.formatOnSave": true,

    "package-lock.json",

    ".git/objects",

  "files.associations": {
    "*.md": "mdx"
